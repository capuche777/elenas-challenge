# Calculadora de comisiones

### Requisitos:
 - Python 3

### Instrucciones:
 - El archivo debe ser ejecutado en terminal
 - El archivo 'orders.csv' debe estar en el mismo folder
 - Al finalizar se recibe un mensaje de confirmación
 - Para la correcta ejecución del programa no se debe remover ninguno de los archivos contenidos ("app.py", "creator.py")

### Ejecutar el archivo
Abrir una términal, navegar hasta el folder que contiene los arvhivos y escribir:
```sh
$ python app.py
```
